# Magnit - Кабельная система передачи энергии

## Описание проекта

**НПО "Лаборатория К"** разрабатывает инновационную кабельную систему **БОГЭКС / Лаб.К-03** для передачи электроэнергии на сверхдальние расстояния с использованием резонансного метода однопроводниковой передачи на токах высокой частоты (ТВЧ).

Проект включает разработку:
- Комбинированного энергетического и оптоволоконного кабеля
- Резонансной системы передачи электроэнергии (РОЭС)
- Подводного телеуправляемого аппарата (ROV) с кабельным управлением
- **Цезиевого магнитометра** на базе STM32 с интеграцией BlueOS и AI Hailo 8L

---

## Основные фичи и возможности

### 1. Кабельная система Лаб.К-03

| Характеристика | Значение |
|----------------|----------|
| Диаметр кабеля | от 1.5 мм |
| Вес | 3-7 кг/км |
| Передаваемая мощность | до 110 кВт |
| Дальность передачи | до 9000 км |
| Количество проводников | 2 электрических + 2 оптоволоконных |

**Ключевые особенности:**
- Сверхлегкий вес (в 2 раза легче аналогов П-274М и П-268)
- Комбинированная передача энергии и данных по одному кабелю
- Бронированная защита оптоволокна
- Работоспособность при экстремальных температурах (до 750°C в течение 3 часов)
- Защита от грызунов, УФ-излучения, механических повреждений

### 2. Резонансная система передачи энергии (РОЭС)

**Принцип работы:** Однопроводниковая линия с высокочастотным резонансным трансформатором Тесла, использующая стоячие волны для передачи энергии.

**Преимущества:**
- Отсутствие коротких замыканий в однопроводниковой линии
- Автоматическое отключение при обрыве линии (изменение частоты)
- Экономия проводниковых материалов
- Пожаробезопасность
- Экологическая безопасность
- Не требует отвода земель под ЛЭП

### 3. Измерительные возможности (на основе оптоволокна)

- **Температура** — контроль температуры среды вдоль всего кабеля
- **Сейсмоактивность** — обнаружение вибраций и движения
- **Выделение газов** — контроль утечек (метан, бутан)
- **Охрана периметра** — обнаружение приближения объектов
- **Контроль места среза** — определение точки повреждения кабеля

### 4. Подводный аппарат (ROV)

- Телеуправление по кабелю
- Видеопередача высокой четкости (HD-SDI)
- Управление по протоколу Pelco-D
- Стереокамеры и 360° съемка
- Сонар и навигация
- Манипуляторы

### 5. Цезиевый магнитометр (BlueOS Extension)

**Новое направление проекта:** Высокоточный цезиевый магнитометр для геофизических исследований с интеграцией в экосистему BlueOS.

| Характеристика | Значение |
|----------------|----------|
| Принцип работы | Оптическая накачка Cs-133 |
| Диапазон измерений | 20 000 - 100 000 нТл |
| Разрешение | 0.001 нТл |
| Частота измерений | 1-1000 Гц |
| Гиромагнитное отношение | 3.498 Гц/нТл |
| Микроконтроллер | STM32 |
| AI-ускоритель | Hailo 8L (13 TOPS) |

**Ключевые возможности:**
- Интеграция с **BlueOS** (Blue Robotics)
- AI-обработка через **Hailo 8L** для обнаружения аномалий
- Шумоподавление нейросетью в реальном времени
- Классификация аномалий (металл, геология, инфраструктура)
- REST API и WebSocket для потоковых данных
- MAVLink телеметрия

**Структура Python модуля:**
```
src/magnetometer/
├── __init__.py           # Экспорт классов
├── config.py             # Конфигурация (PEP8)
├── stm32_driver.py       # Драйвер связи с STM32
├── cesium_magnetometer.py # Основной класс магнитометра
├── hailo_processor.py    # AI-обработка Hailo 8L
├── blueos_extension.py   # Расширение BlueOS
├── main.py               # Точка входа
└── requirements.txt      # Зависимости
```

**Запуск:**
```bash
# Установка
pip install -e src/

# Запуск
magnit-magnetometer --port /dev/ttyUSB0 --sample-rate 100

# С отключённым AI
magnit-magnetometer --no-hailo
```

**API Endpoints:**
```
GET  /v1/status      - Статус устройства
GET  /v1/reading     - Текущее измерение
GET  /v1/readings    - Буферизованные измерения
GET  /v1/statistics  - Статистика измерений
POST /v1/calibrate   - Запуск калибровки
POST /v1/config      - Обновление конфигурации
WS   /v1/ws          - WebSocket поток данных
```

```
Цель: Разработка оборудования резонансной системы питания
      подводного аппарата на расстояние до 1240 м

Состав системы:
├── Передающий преобразователь
├── Линия электропередачи
└── Обратный преобразователь

Технические параметры:
┌─────────────────────────────────────────────────────────────┐
│ Напряжение питания передатчика    │ ~220В, 50Гц            │
│ Суммарная мощность нагрузки       │ 200 Вт (с резервом)    │
│ Напряжение в линии                │ 900 В                  │
│ Частота в линии                   │ 1-8 кГц                │
│ Выходное напряжение приёмника     │ 22-36 В DC             │
│ Максимальная длина линии          │ 1240 м                 │
│ Габариты приёмника                │ ø60 × 200 мм           │
│ Габариты передатчика              │ ø190 × 250 мм          │
└─────────────────────────────────────────────────────────────┘
```

### Электрические характеристики коаксиального кабеля

- Волновое сопротивление: 75 ± 3.5 Ом
- Электрическая ёмкость: 55 пФ/м
- Коэффициент укорочения длины волны: 1.22
- Сопротивление связи: ≤15 мОм/м
- Скорость распространения волны: 82%

## Целевые рынки применения

1. **Нефтегазовая отрасль** — безопасная передача энергии, контроль утечек газа
2. **Полевые кабельные системы** — легкие системы связи для военных и МЧС
3. **Научное оборудование** — полевые исследования, подводные аппараты
4. **Охранные системы** — периметральная охрана с сейсмодатчиками
5. **Удалённое электроснабжение** — тундра, тайга, горы, острова
6. **Аэрокосмические системы** — бортовые кабельные системы
7. **Подводная техника** — ROV, подводные станции
8. **Железнодорожный транспорт** — системы навигации и связи
9. **Пожарная безопасность** — контроль температуры, автоматическое оповещение
10. **Телекоммуникации** — защищённые линии связи

- Карбид кремния (SiC) для высоковольтных генераторов
